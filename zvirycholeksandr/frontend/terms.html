1. Опис послуг

zvirycholeksandr.com.ua надає послуги з розробки веб-сайтів для малого бізнесу, включаючи:

  • Лендінги для місцевих спеціалістів
  • Сайти-візитки для фахівців та підприємців
  • Онлайн-меню для закладів харчування

2. Процес замовлення

Замовлення здійснюється через форму на сайті. Після отримання заявки:

  • Відповідаю протягом 24 годин (у робочі дні)
  • Уточнюю деталі та вимоги до проєкту
  • Надаю технічне завдання та фінальну ціну
  • Після погодження розпочинаємо роботу

3. Вартість та оплата

Вартість послуг визначається індивідуально для кожного проєкту і залежить від:

  • Типу та складності сайту

  • Строків виконання

Оплата здійснюється: 50% передоплата, 50% після передачі готового сайту.

4. Строки виконання

Стандартний строк розробки — 5–14 робочих днів залежно від складності проєкту. Точні строки погоджуються в договорі.

Клієнт зобов'язується:

  • Надавати достовірну інформацію для розробки

  • Оплачувати послуги у погоджені строки

Надається 14-денна безкоштовна гарантія на виправлення помилок у розробленому функціоналі з дня передачі проєкту. Після гарантійного строку технічна підтримка надається на платній основі — разово або у форматі щомісячного супроводу.
